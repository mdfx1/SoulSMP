# Häufig gestellte Fragen

## Was ist SoulSMP?
SoulSMP ist ein Deutscher Minecraft SMP, auf dem stets neue Updates kommen.  
Außerdem haben wir einen Discord, welcher zum Chatten genutzt werden kann.  

## Wie erstellt man ein Ticket?
Im ⁠🎫・ticket-support drückst du auf **Ticket erstellen** und anschließend füllst du im folgenden  
Fenster die Frage aus, wieso du dein Ticket erstellst. Beantworte die Frage so präzise  
wie möglich. Bleibe bitte im Verlauf deines Tickets stets sachlich und  
halte dich an das Thema deines Tickets.  

##  Darf man andere Spieler ohne Grund töten?
Nein. Genaueres findet ihr im ⁠[Regelwerk](./regelwerk).  

## Kann man mit Cracked Accounts auf den Server?
Nein, das Betreten des Servers mit einem Account, der nicht der offiziellen  
Java- und/oder Bedrock-Version angehört, ist nicht möglich.  

##  Auf welcher Version ist der Server und wie ist die IP?
Ihr erreicht den Server über die IP **soul.smpmc.eu** mit allen Versionen von **1.20** aufsteigend.  
Empfohlen ist jedoch die Version **1.20.4**, um alle Funktionen lagfrei nutzen zu können.
